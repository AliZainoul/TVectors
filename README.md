# Template Version of Class Vector

A template version of class Vector

## Installation & Usage
Make sure you have G++ Compiler first, then type in Terminal :

  g++ -std=c++11 -c error.cpp Main_Vector.cpp

  g++ -o main error.o Main_Vector.o

  ./main
